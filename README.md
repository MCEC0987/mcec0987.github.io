# Aaa导航

一个基于 Vue 3 和 Vuetify 3 的现代化导航页。

## 主要功能

-  **轮播图**：自动轮播，点击跳转。
-  **多引擎搜索**：支持 Bing、百度、Google、搜狗。
-  **分类管理**：可折叠分类，点击筛选链接。
-  **收藏功能**：长按或点击星标收藏/取消收藏。
-  **主题切换**：支持浅色/深色模式及多种主题色。
-  **响应式设计**：适配手机、平板、桌面。
-  **布局设置**：支持单列/双列布局。
-  **数据管理**：支持导出/导入数据，本地存储配置。

## 快速开始

直接打开 `index.html` 即可使用。
